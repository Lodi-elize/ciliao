## 四、功能概述

次聊是一款用于验证 NFC 快速加好友和一对一聊天流程的移动端社交聊天软件。用户可以通过用户名密码或手机号 mock 验证码注册登录，设置昵称、头像和个性签名，搜索其他用户并发送好友申请。软件支持通过 NFC 标签读取 `nfcchat://add-friend?userId=...` 格式的邀请信息，自动识别目标用户并发起好友申请；在无 NFC 设备或模拟器环境下，也支持手动 invite fallback。

好友申请被对方同意后，系统自动建立双向好友关系并更新通讯录。用户可进入一对一聊天界面发送文本消息，后端负责消息保存、会话维护和 WebSocket 实时推送。软件还记录 NFC 读取事件，便于后续统计、追踪和排查设备读卡链路问题。

## 五、主要模块

| 模块 | 功能 |
| --- | --- |
| 账号认证模块 | 用户名注册、手机号 mock 注册、登录、自动登录、退出登录、修改密码 |
| 用户资料模块 | 昵称、头像、个性签名、个人信息维护 |
| 好友搜索模块 | 按 username、userId、手机号搜索用户 |
| 好友申请模块 | 发送、接收、同意、拒绝好友申请，建立双向好友关系 |
| NFC 邀请模块 | 读取 NDEF URI/Text 记录，解析邀请用户，记录读卡事件 |
| 通讯录模块 | 展示好友列表，维护联系人关系 |
| 聊天模块 | 一对一文本消息发送、聊天记录读取、会话状态刷新 |
| 实时推送模块 | WebSocket 消息和联系人状态推送 |
| 数据持久化模块 | 用户、凭证、会话、消息、联系人、好友申请、NFC 事件存储 |
